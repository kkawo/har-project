# 离线数据采集说明

## 硬件准备

1. ESP32-S3 + GY-521 (MPU6050) Shield 焊接板
2. 充电宝供电（不连电脑）
3. **传感器佩戴**：腰带部位，绑紧不滑动
   - 方向不重要，但**两人（S01、S02）戴法必须一致**
   - 推理时也按同样方式佩戴

## 刷固件

1. 用 Thonny 打开 `firmware/offline_logger/main.py`
2. 修改第 44 行 `SUBJECT`：
   - 你 → `SUBJECT = "S01"`
   - 李宝平 → `SUBJECT = "S02"`
3. 点击 Run 或保存到 ESP32

## 灯光信号

| 灯光 | 含义 |
|------|------|
| 🔵 蓝 | 静坐 |
| 🔷 青 | 站立 |
| 🟢 绿 | 步行 |
| 🟡 黄 | 跑步 |
| 🟠 橙 | 上楼 |
| 🟣 紫 | 下楼 |
| 🔴 红 | 跌倒 |
| 暗闪（同色） | 倒计时中，准备 |
| 常亮（同色） | **正在录制，保持动作** |
| ⚪ 白闪 1 次 | 本 trial 完成 |
| ⚪ 白闪 3 次 | 全部 21 trial 完成 |

## 采集流程

每人 7 种活动 × 3 次 = 21 trial，约 11 分钟。

### 节奏

1. **颜色慢闪 3 次**（3 秒）→ 看颜色，知道下一个动作
2. **暗闪倒计时**（8 秒，第一个 trial 3 秒）→ 摆好姿势
3. **常亮** → **开始做动作**，灯灭就停
4. **白闪 1 次** → 休息 8 秒，下一个动作
5. 重复 21 次 → **彩虹 + 白闪 3 次** → 结束

### 各动作要求

| 动作 | 时长 | 说明 |
|------|------|------|
| 静坐 | 30s | 正常坐姿，不晃动 |
| 站立 | 30s | 自然站立，可以微微换重心 |
| 步行 | 30s | 匀速直走，自然摆臂 |
| 跑步 | 30s | 慢跑或快跑，稳定节奏 |
| 上楼 | 30s | 一步一步上楼梯 |
| 下楼 | 30s | 一步一步下楼梯 |
| 跌倒 | 15s | 向前或侧倒（软垫），然后站起来 |

> 做动作时尽量**自然连续**，不要突然停顿或加速。跌倒后自己站起来即可。

## 下载数据

采集完成后连电脑，在 Thonny 中下载所有 CSV 文件（21 个）。

文件命名：`{SUBJECT}_{动作}_{trial号}.csv`
例如：`S01_walk_02.csv` = S01 第 2 次步行

把 CSV 放到 `data/raw/real/S01/` 和 `data/raw/real/S02/`。

## 重新训练模型

```bash
python src/preprocess.py          # 预处理
python src/features.py            # 特征提取
python src/d11_export_model.py    # 导出模型参数到 firmware/realtime_inference/model_params.py
```

然后重新刷 `main_wifi.py` + `model_params.py` 到 ESP32 即可实时推理。
