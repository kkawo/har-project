### 二、项目背景与目标

#### 2.1 项目背景

人体活动识别（Human Activity Recognition, HAR）是可穿戴计算与普适计算领域的核心问题，广泛应用于健康监护、运动分析、跌倒检测、智能家居等场景。本项目利用低成本 MEMS 惯性传感器（MPU6050 加速度计+陀螺仪）与磁力计（HMC5883L），构建从原始信号采集到活动分类的完整模式识别系统。

#### 2.2 项目目标

**总体目标**：独立完成「自己采数据 → 自己做预处理与标定 → 自己设计特征 → 自己选模型调参 → 用严谨实验证明识别精度」的完整闭环。

**量化指标**：
- 6 类日常活动整体识别准确率 ≥ 80%（被试独立 LOSO 评估）
- 完成 ≥2 项传感器标定，并量化标定对识别精度的增益
- 对比 ≥5 种分类器，给出严谨的统计评估

### 三、问题形式化

#### 3.1 输入空间

| 要素 | 定义 |
|------|------|
| 传感器 | MPU6050（3 轴加速度 + 3 轴陀螺仪）+ HMC5883L（3 轴磁力计） |
| 采样率 | 50 Hz（满足人体运动 ≤20 Hz 的采样定理要求） |
| 佩戴位置 | 腰部（固定绑带） |
| 信号维度 | 9 通道（ax, ay, az, gx, gy, gz, mx, my, mz） |

#### 3.2 输出空间（类别集）

| 编号 | 活动类别 | 难度 | 备注 |
|------|----------|------|------|
| 0 | 静坐 | 基础 | 低强度、低方差 |
| 1 | 站立 | 基础 | 与静坐易混淆（微小姿态变化） |
| 2 | 平地步行 | 基础 | 周期性信号，特征明显 |
| 3 | 跑步 | 基础 | 高强度周期性信号 |
| 4 | 上楼 | 基础（易混淆） | 与下楼考验频域特征判别力 |
| 5 | 下楼 | 基础（易混淆） | 与上楼考验频域特征判别力 |
| 6 | 跌倒（进阶） | 进阶 | 高冲击、漏检代价极高 |

#### 3.3 决策函数

- **最小错误率决策**（默认）：选择后验概率最大的类别
- **最小风险决策**（进阶）：为跌倒漏检设计代价矩阵，牺牲整体准确率换取跌倒召回率

#### 3.4 样本定义

- 滑动窗口长度：2.56 s（128 采样点 @ 50 Hz）
- 重叠率：50%
- 每窗口 = 1 个样本，窗口标签采用多数表决

### 八、风险预判与应对

| 风险 | 影响 | 应对策略 |
|------|------|----------|
| 标定操作不当致数据质量差 | 识别率上限被压低 | D2 严格按六位置法操作，静置时确保传感器完全静止 |
| 被试不足 | 模型泛化能力无法验证 | 组内成员互为被试（≥3人），提前约定采集时间 |
| 数据泄漏（同被试跨训练/测试集）| 识别率虚高，验收不通过 | 严格按 subject_id 划分，Pipeline 封装防泄漏 |
| 类别混淆（上楼 vs 下楼）| 拉低整体准确率 | 侧重频域特征（步频差异）+ 气压计辅助 |
| PCB 下单超时 | 无法冲 80+ | D1 启动 KiCad，D3 中午前完成下单 |
| 个别类别数据量不足 | 类别不平衡 | 采集时严格控制每类 ≥3 次 × ≥60s |
