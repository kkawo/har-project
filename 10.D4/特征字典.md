## 特征计算公式

### 时域特征
| 特征 | 公式 |
|------|------|
| mean | $\mu = \frac{1}{N}\sum x_i$ |
| std | $\sigma = \sqrt{\frac{1}{N}\sum(x_i - \mu)^2}$ |
| var | $\sigma^2$ |
| rms | $\sqrt{\frac{1}{N}\sum x_i^2}$ |
| peak_to_peak | $\max(x) - \min(x)$ |
| skew | $\frac{1}{N}\sum\left(\frac{x_i - \mu}{\sigma}\right)^3$ |
| kurtosis | $\frac{1}{N}\sum\left(\frac{x_i - \mu}{\sigma}\right)^4 - 3$ |
| zero_cross_rate | $\frac{1}{N}\sum \mathbb{1}[\text{sign}(x_i - \bar{x}) \neq \text{sign}(x_{i-1} - \bar{x})]$ |
| sma | $\frac{1}{N}\sum |x_i|$ |
| autocorr_lag1 | $\frac{\sum(x_i - \mu)(x_{i+1} - \mu)}{\sum(x_i - \mu)^2}$ |

### 频域特征
| 特征 | 公式 |
|------|------|
| dominant_freq | $\arg\max_f |X(f)|$ |
| spectral_centroid | $\frac{\sum f \cdot |X(f)|}{\sum |X(f)|}$ |
| spectral_entropy | $-\sum P(f) \log P(f)$  where $P(f) = |X(f)|^2 / \sum|X(f)|^2$ |
| spectral_bandwidth | $\sqrt{\frac{\sum (f - f_c)^2 |X(f)|}{\sum |X(f)|}}$ |
| spectral_flatness | $\frac{\sqrt[N]{\prod P(f)}}{\frac{1}{N}\sum P(f)}$ |
| spectral_rolloff | $f$ such that $\sum_{0}^{f} |X(k)|^2 = 0.85 \cdot \sum |X(f)|^2$ |
